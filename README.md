# Development

1. Clone the repo
```
git clone https://github.com/JPI-Tech/trshmag.git
```

2. Install packages
```
cd website
npm install
```

3. Run the UI server
```
npm run dev
```

4. Open a new Terminal tab and run the web server
```
node server
```

5. UI should open in the browser at `localhost:3000`
